# 2D-Discrete-Wavelet-Transform---Haar-Wavelets
This Program Implements 2 dimensional discrete wavelet transform
The wavelet used to implement discrete wavelet transform is Haar wavelets
OpenCV is also required to run this program


Steps to run the program
-To compile the example file, over the terminal type './compileCV exampefile.cpp'
-Execute the output file by typing './examplefile'
